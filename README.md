# Student Expense Tracker
A Python-based command-line application designed to help students manage their weekly allowance. This program allows users to track their spending across various categories and provides real-time feedback on their budget status.

**Key Features:**

* Budget Overview: Set a weekly budget and track total spending.
* Categorized Logging: Organize expenses into five hardcoded categories (Food, Transportation, Mobile/Internet, School Supplies, and Entertainment).
* Smart Alerts: Automatically flags any single purchase that exceeds 25% of the total weekly budget with a "High Expense Alert."
* Summary Report: Generates a formatted log at the end of the session showing the remaining balance and a status message based on spending habits.

**How to Run:**
1. Ensure you have Python installed.
2. Run the script using: python midterm_solution.py
3. Follow the on-screen prompts to enter your name, budget, and up to four expense entries.
